# Go User API

A RESTful API built using Go to manage users with Date of Birth (DOB) and dynamically calculated age.

## Tech Stack
- Go (Golang)
- Fiber (Web framework)
- PostgreSQL
- SQLC (Type-safe DB access)
- go-playground/validator
- Uber Zap (logging)

## Features
- Create user with name and DOB
- Fetch user by ID with dynamically calculated age
- Update user
- Delete user
- List all users
